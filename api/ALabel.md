# ALabel
> **Extends**: `AComponent`

라벨 컴포넌트

<br/>

## Methods

### getText()

라벨의 텍스트를 반환한다.

* **Usage**: 
```js
var result = label.getText();
```

<br/>

### setHtml( html )

라벨에 html태그를 세팅한다.

* **Parameters**: 
	* **`html`** {String} html 태그

* **Usage**: 
```js
var tag = 'html tag....';
label.setHtml(tag);
```

<br/>

### setText( text )

텍스트를 세팅한다.

* **Parameters**: 
	* **`text`** {String} 텍스트

* **Usage**: 
```js
label.setText('텍스트');
```

<br/>

### setTextAlign( align )

텍스트의 text-align 속성을 세팅한다.

* **Parameters**: 
	* **`align`** {String} text-align css value

* **Usage**: 
```js
label.setTextAlign('center');
```
