# AIndicator
> **Extends**: `AFloat`

AIndicator

<br/>

## Methods

### createSpan()

Indicator를 초기화 합니다. (Show함수만 호출하면 내부적으로 createSpan를 자동으로 호출합니다.)

<br/>

### hide()

인디케이터를 숨깁니다.

<br/>

### hide()

인디케이터를 숨긴다.

* **Usage**: 
```js
AIndicator.hide();
```

<br/>

### show()

인디케이터를 표시합니다.

<br/>

### show()

인디케이터를 표시한다.

* **Usage**: 
```js
AIndicator.show();
```
